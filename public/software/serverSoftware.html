Home Hub

Multi-Page, component based smart home hub

The current era, the Home Hub era. This is the website you are currently viewing and is the most advanced to date. Built off of the project portfolio design it was not meant to be its own era however the single page application (SPA) approach of the project portfolio site lacked features that would benefit the smart home home. This lead to a major functional rework of the site where it was decided to overhaul the appearance as well to a more modern design. The main features of this version are listed below:

  • Custom Component Based Framework
  • RESTful API
  • HTTPS Security
  • More modern GUI
  • Smart Home Device Control
  • Secure Log In
  • Git and Github






Project Portfolio

SPA, RESTful API project portfilio with expecting smart home update

This era ended in May 2020 and was the major influencer of the current era. The previous eras were soley focused on creating smart home devices from the ground up and integrating them together however the project portfolio era changed direction and focused on a website showcasing these projects. The goal was eventually to integrate all the smart home features from previous projects however the portfolio became the main focus as something to show to potential employers. In the end, most of this era focused on website and server API development The main features of this version are listed below:

  • RESTful API
  • HTML, JQuery, CSS, Javascript, NoSQL, AJAX
  • Single Page Application (SPA)
  • NoSQL - MongoDB via Mongoose
  • Interactive Slide Shows
  • Custom AJAX queries for content manipulation
  • Git and Github

Fanshawe College Capstone

3 IoT Devices

This was the start of the smart home fascination that has fueled me for the last couple years. As part of the Fanshawe College electrical and electronics engineering program, we had a Capstone project where I chose to develop 3 smart home devices:

  • Smart Door Lock
  • Smart Light Switch
  • Smart Power Bar

The main features of this version are listed below:

  • Linux based server
  • NodeJS for serverside code
  • Websockets for wireless network based LAN control
  • Raspberry Pi Zero W
  • 3D designed and printed components
